# Installation

## Requirements

- Python 3.9 or later
- NumPy, Matplotlib, Pandas (installed automatically)

## From PyPI

```bash
pip install evas-sim
```

## From Source

```bash
git clone https://github.com/Arcadia-1/EVAS.git
cd EVAS
pip install -e ".[dev]"
```

## Verify

```bash
evas list
```

You should see the 5 bundled example groups printed.
